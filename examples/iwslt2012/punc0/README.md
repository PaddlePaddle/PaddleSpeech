# Punctuation Restoration with IWLST2012-Zh

## Get Started
### Data Preprocessing
```bash
./run.sh --stage 0 --stop-stage 0
```
### Model Training
```bash
./run.sh --stage 1 --stop-stage 1
```
### Testing

```bash
./run.sh --stage 2 --stop-stage 2
```
### Punctuation Restoration
